Gestión de la Configuración del Software

Descripción

La gestión de la configuración del Software es un proceso en donde se identifican, monitorean y almacenan todos los artefactos de un proyecto de Software.

Características


  • Primer objetivo

    Integridad de la versión actual

    Son cambios de un item de configuración para que sea incorporado en la versión acutal del software y estos cambios son almacenados en un repositorio para que sean recuperados.

  • Segundo objetivo

    Integridad de sobre-escritura

    Se trata de que los desarrolladores pueden trabajar al mismo tiempo en los items de configuración sin afectar a los demás.

  • Tercer objetivo

    Deshacer

    Es devolver a una versión anterior cuando se hace un cambio incorrecto.

  • Cuarto objetivo

    Recuperación de desastres

    Es guardar versiones previas de un software para poder volver a esas versiones en caso de ser necesario.

Actividades en el Plan de Gestión de Configuración del Software


Identificación de la Configuración

Conisiste en identificar los items de configuración y un item de configuración es cualquier artefacto que puede tener modifiaciones durante el proyecto de software.

Items base

Es un artefacto o código fuente que pasa por varias modificaciones hasta volverse estable y en donde se puede usar en desarrollos posteriores.

Número de versión Major.Minor.Patch

El Patch version cambia cuando se solucionan errores particulares en el software. El Minor versión cambia cuando agregan nuevas funcionalidades y manteniendo compatibilidad hacia atrás. El Major Version cambia cuando se agregan nuevas funcionalidades pero no son compatibles hacia atrás.

Conclusión

Podemos concluir que la gestión de la configuración es importante porque nos permite navegar hacia el pasado para recupera una versión funcional o para mirar como se configuro esa versión. También, nos permite identificar cuales ítems necesitan ser modificados hasta que satisfacen la funcionalidad del software. Finalmente, nos permite organizar y clasificar las versiones pasadas para encontrar los más fácilmente.
